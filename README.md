An infinite, AI-driven graphic adventure game where every choice, location, and item is dynamically generated by Google's Gemini models. Step into any world you can imagine and let the AI craft the narrative and visuals in real-time.

## ✨ Features

- **🧠 Dynamic Narrative Engine:** Powered by **Gemini 3.1 Flash Lite**, the game generates unique scenarios, characters, and outcomes based on your actions.
- **👁️ Real-time Scene Visualization:** Uses **Gemini 2.5 Flash Image (Nano Banana)** to generate unique, atmospheric images of your current location.
- **🎒 Interactive Inventory:** Collect items dynamically generated by the AI and use them to solve puzzles or overcome obstacles.
- **🌍 Infinite Worlds:** Start an adventure in a cyberpunk city, a medieval fantasy realm, a post-apocalyptic wasteland, or any custom setting you type.
- **⚡ Fast & Responsive:** Built with React, Vite, and Tailwind CSS for a smooth, modern user experience.

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS, Lucide React (Icons)
- **AI Integration:** `@google/genai` SDK
- **Models Used:**
  - `gemini-3.1-flash-lite-preview` (Text, Logic, and State Management)
  - `gemini-2.5-flash-image` (Scene Image Generation)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Google Gemini API Key (Get one at [Google AI Studio](https://aistudio.google.com/))

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/ai-text-adventure.git
   cd ai-text-adventure
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your Gemini API key. *(Note: If you are running this in a cloud environment like AI Studio, the key is injected automatically).*
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:3000` to start your adventure.

## 🎮 How to Play

1. **Choose a World:** Enter a theme or setting for your adventure (e.g., "A haunted Victorian mansion" or "A neon-lit cyberpunk alley").
2. **Choose an Art Style:** Select the visual style for the generated images (e.g., "Pixel Art", "Photorealistic", "Watercolor").
3. **Make Choices:** Read the scene description and choose one of the available actions.
4. **Manage Inventory:** Keep an eye on the items you collect; they might unlock new paths or save you from danger!
